Web Applications

Exam Scheduler

There is a huge disconnect between college students and professors when it comes to understanding each other's schedules. This leads to increasing levels of stress between both parties and decreasing levels of performance inside the classroom. This web app allows a professor to see when all of the students in his/her classes have exams so he/she can schedule exams with empathy.
